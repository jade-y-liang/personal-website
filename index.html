Projects

Here are some of the projects that I've completed.

No-Plan Pantry

A recipe recommendation web application that recommends recipes from allrecipes.com based on user inputted ingredients. The goal of this project is to reduce food waste by inspiring users to use their leftover ingredients to make a delicious meal! This web application also generates AI-generated recipes based on user-inputted ingredients.

Redfin Housing Analysis

In this ongoing project, I am responsible for cleaning and preprocessing a comprehensive dataset from Redfin, aimed at understanding the dynamics of the housing market. The primary goal is to develop a predictive model that can accurately forecast housing prices based on various influencing factors.

Image Classification of Cats and Dogs

This project leverages the power of deep learning to accurately differentiate between images of cats and dogs. Using TensorFlow and Keras, a series of increasingly complex models were developed, starting from basic convolutional neural networks to sophisticated architectures employing data augmentation and transfer learning. Each model iteration was evaluated for performance improvements, culminating in a robust model that demonstrates an impressive accuracy of 96.73% on test data.

Fake News Classification

This project explores the development of a machine learning model capable of identifying fake news articles using TensorFlow and Keras. The model is trained on a dataset that includes titles and texts of news articles, along with labels indicating whether the news is fake or real. The primary objective is to evaluate the effectiveness of using article titles, texts, or both to predict the legitimacy of news. This involves pre-processing text data, employing text vectorization, and building multiple neural network architectures to compare their performance.

Academic Performance Analysis

This project was completed as part of my final coursework in Stats 101A at UCLA. The goal was to predict students' academic performance indices using a dataset of 10,000 entries from Kaggle. This analysis is crucial for understanding the factors that influence academic success and can help educators tailor their approaches.
